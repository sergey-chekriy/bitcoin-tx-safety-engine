# Security Policy

This repository is intended for open-source Bitcoin developer tooling.

## Scope

Please report security issues related to:

- unsafe transaction analysis behavior
- misleading analysis results that could create material risk
- dependency or supply-chain concerns in the published package
- documentation errors that could reasonably lead to unsafe usage

## Disclosure

Please do not open a public issue for suspected security-sensitive problems. Instead, report the issue privately to Sergey Chekriy at chekriy@gmail.com.

## Important limitation

This project is intended to provide transaction safety analysis and warnings. It is not a guarantee that a transaction is safe, valid, standard, or appropriate for broadcast under all conditions.
